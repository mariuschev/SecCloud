Foire Aux Questions (FAQ)

Comment puis-je créer un compte ?

Pour créer un compte, cliquez sur "S'inscrire" dans le menu et remplissez le formulaire d'inscription.

Comment réinitialiser mon mot de passe ?

Sur la page de connexion, cliquez sur "Mot de passe oublié" et suivez les instructions.

Comment contacter le support ?

Vous pouvez nous contacter via email à support@monsite.com.

Comment supprimer mon compte ?

Rendez-vous dans les paramètres de votre compte et cliquez sur "Supprimer mon compte".

Quels services puis-je déployer sur la plateforme ?

Vous pouvez déployer des services comme WordPress, PostgreSQL et un stockage compatible S3 directement depuis votre tableau de bord.

Comment déployer WordPress ?

Allez dans le tableau de bord, cliquez sur "Déployer WordPress" et suivez les instructions. Vous pourrez ensuite accéder à votre site WordPress via l'URL fournie.

Comment déployer une base de données PostgreSQL ?

Dans le tableau de bord, sélectionnez "Déployer PostgreSQL". Une fois le service lancé, les informations de connexion vous seront affichées.

Comment déployer un service de stockage S3 ?

Depuis le tableau de bord, cliquez sur "Déployer S3". Vous obtiendrez ensuite les identifiants d'accès pour utiliser le stockage.
